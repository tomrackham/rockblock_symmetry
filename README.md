# RockBlock Symmetry

A modern HTML5 remake of the classic 2000-era puzzle game by Tom Rackham. 

**Objective:** Push all the coloured rocks until they convert into walls. The level is complete when the final arrangement of walls forms a perfectly symmetric layout across all four axes (horizontal, vertical, and both diagonals).

## Features
- Classic 32x32 retro pixel art and gameplay mechanics
- Persistent level progress and saved board states
- Modern interface with visual symmetry guides
